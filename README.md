# Проект: Место 

### Описание
Проект место - сайт, который разработан при помощи технологий HTML, CSS и JavaScript


**Функциональность**
* Flex-box
* JavaScript
* Position
* Адаптивная верстка

```sh
git clone https://github.com/aKrivonogova/mesto
```
```sh
cd mesto
```

```sh
npm i webpack --save-dev
```

```sh
npm run build
```
